# AgenticFish marketplace

Claude Code plugin marketplace for [AgenticFish](https://github.com/AgenticFish) plugins.

## 安装

```bash
# 在 Claude Code 里
/plugin marketplace add AgenticFish/marketplace
```

## 可用插件

| 插件 | 说明 | 仓库 |
|------|------|------|
| [composir](https://github.com/AgenticFish/composir) | 科普文写作工作流 | `AgenticFish/composir` |

## 安装单个插件

```bash
/plugin install composir@agenticfish
```
